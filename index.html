Alize

“Combining my love of music and programming, I created a music streaming and download/upload service with audio player visualization and the ability to share, play and create playlists. Using postgres, I had 3 models, all many to many relationships. The visualization on the audio players used Web API, Audiocontext to analyze the track, thereafter plotting the data onto a HTML5 canvas was most problematic I ran into Cors restrictions and only realised on the final day that an audio node can only be connected to 5 audio sources at one time.”

Used: Sinatra framework, Ruby, JavaScript, psql, bcrypt, carrierwave, html5 canvas, web audio API audiocontext

Crossroads

“I Created this app as a solution to integrate, into our society, the many refugees arriving into our country as well as connecting those that want to help with those that need help. We used node.js as a framework, working in a team of three. After extensive planning, a colleague and I pair-coded tests for our back-end API using Mocha and Chai. I set up the API including authorization using bcrypt and JWT. I integrated the Stripe API into the app, assigning each donation to a particular refugee. For this, app it was necessary to have two different authenticated users, a charity and a potential host. The different roles on login was solved by having one user model, and storing the role, that was a hidden field on the registration form, into local storage through the payload on login and registartion. This method made it very easy to control accessibility on the front-end.”

Used: Node.js, JavaScript, TDD, Mocha, Chai, Google maps API, JWT, Bcrypt, Guardian API, mongoose, Ajax, JQuery, Bluebird

Dose

“I created a dashboard in which users can add widgets to their board. The concept was to add a 'dose' of each source to their board, in turn, eliminating the mess of having many tabs open at once. I used many third party API's in order to create my tools and felt I really came out of this project with a fantastic understanding of AngularJS. I used websockets for the 'notes' widget, as well as angular-gridster and an image uploader, which allowed the user to customise the own dashboard choosing the background, the layout and the widget sizes.”

Used: Node.js, AngularJS, sockets.io, TFL api, crowd-scores api, guardian api, football-data api, openweather api, JWT, Oauth, S3, multer, angular-gridster, satellizer, ng-uploader, bluebird
