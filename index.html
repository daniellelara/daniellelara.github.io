Alize

“Combining my love of music and programming, I created a music streaming and download/uplload service with audio player visualization and the ability to share, play and create playlists. Using postgres, I had 3 models, all many to many relationships. The visualization on the audio players which used a JavaScript API Audiocontext to analyze the track, thereafter plotting the data onto a HTML5 canvas was most problematic I ran into Cors restrictions and only realised on the final day that an audio node can only be connected to 5 audio sources at one time.”

Used: Sinatra framework, Ruby, JavaScript, psql, bcrypt, carrierwave, html5 canvas, web audio API audiocontext
